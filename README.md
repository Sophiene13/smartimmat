# SmartImmat

SaaS dédiée aux professionnels de l'automobile — Projet de certification DWWM (Développeur Web et Web Mobile)

## Description

SmartImmat est un CRM développé pour les professionnels de l'automobile et motocycles:garages, concessions, agents. L'application permet de gérer les dossiers d'immatriculation qu'il s'agisse de leurs clients ou de leur propre flotte.
## Stack Technique

### Backend
- **Framework** : Express JS
- **Base de données** : Architecture hybride
- **Architecture** : API REST (MVC)

### Frontend
- **Structure**: HTML
- **Style**: CSS
- **Approche**: Full vanilla JS(sans framework)
